# Strand Surf Club — Website Scaffold

Landing site for **Strand Surf Club**, Cape Town's surf instruction club on the Western Cape's best breaks.

Built by [Linfy Tech](https://linfy.tech), Strand, Cape Town.

---

## Pages / Sections

| Section | Anchor |
|---|---|
| Home (Hero) | `#home` |
| About | `#about` |
| Book a Lesson | `#book-a-lesson` |
| Contact | `#contact` |

---

## Tech Stack

- Pure **HTML5 / CSS3 / Vanilla JS** — zero dependencies, zero build step
- **[Fontshare](https://www.fontshare.com/)** — Boska (display) + Satoshi (body)
- Full **CSS custom property** design token system
- **Dark mode** via `[data-theme]` on `<html>`
- Mobile-first layout, verified at 375px
- Scroll-reveal animations via `IntersectionObserver`

---

## Development

Open `index.html` in any modern browser — no build step or server required.
